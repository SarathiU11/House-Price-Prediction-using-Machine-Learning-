## 🧠 Overview
This project is a supervised machine learning regression problem, where the goal is to predict the selling price of a house based on several independent variables.

---

## 📊 Dataset

- **Attributes:** 80+ features including:
  - Lot Area
  - Year Built
  - Number of Bedrooms
  - Garage Area
  - Neighborhood
